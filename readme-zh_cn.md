这是一个基于pygame-ce的音乐播放器。你不能用pygame运行，因为pygame没有`box_blur()`模糊函数。——在命令行输入`pip install pygame-ce`，然后开始使用pygame社区版吧！

**已知问题：程序只能在1600x900下运行，否则会出现缩放问题。**（将在未来解决）
